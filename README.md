# True Progress Legacy

A lightweight UI utility designed specifically for Geometry Dash **v2.2074** and Geode **v4.10.2**.

### 🚀 Features:
* **StartPos Tracking:** Displays your exact starting position percentage.
* **Real-time Progress:** Shows current run percent side-by-side.
* **Clean Format:** `[Start]% - [Current]%` layout replaces the default screen clutter.
